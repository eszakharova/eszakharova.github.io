Experience

  • Internships
    • Google Summer of Code (May - August 2017) with the project ScrapingHub: adding the support for finding and parsing dates in the large texts to dateparser
      Dateparser is a Python library (available for Python 2 and 3) used to parse dates in various formats in different languages to uniform format objects. Aim of my project was to add to dateparser support of search and parsing of date expressions in the large chunks of natural language text. In the result dateparser was supplemented by a function which receives a string in one of the 29 languages currently supported by dateparser and returns substrings which represent found date expressions paired with corresponding datetime.datetime objects. All written code is compatible with both Python 2 and 3 and is covered by tests. (Final Evaluation link).
